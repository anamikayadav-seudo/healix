What is Healix.AI?

Healix.AI is an AI-powered preventive healthcare platform designed to bridge the gap between symptom awareness and specialist consultation. Our system integrates structured machine learning models, explainable AI reasoning engines, and intelligent doctor mapping logic to provide meaningful insights within seconds.

How It Works

1. Enter Symptoms

Users input structured symptom data through a guided interface.

2. ML Prediction

RandomForest classification model predicts probable conditions.

3. AI Explanation

Groq LLaMA generates human-readable reasoning.

4. Specialist Mapping

Disease → Specialty → Recommended Doctor pathway.
